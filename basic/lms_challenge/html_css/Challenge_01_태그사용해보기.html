ISFP의 특징 by. 나무위키

호기심 많은 예술가, 성인군자형

  • 모든 유형 중에서 침대 위에 누워서 아무것도 안하는 시간이 제일 길다.
  • 화가 나면 논리적으로 반박하기보다 침묵하는 편이다.
  • 남에게 싫은 소리 못 하고 속으로 삭힌다.
  • 다른 사람의 부탁을 거절하기 어려워한다.
  • 대체로 남을 잘 믿는 편이다.
  • 첫인상은 차갑고 수줍어 보이나 알고 보면 꽤 순둥이다.
  • 타인의 비판에 대해 개방적이며 잘 수긍한다.
